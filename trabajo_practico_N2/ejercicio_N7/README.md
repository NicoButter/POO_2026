# Ejercicio N7 - Simulación de impresora

Este ejercicio simula el funcionamiento de una impresora con gestión de tinta, papel, conexión y estado en línea, utilizando programación orientada a objetos y manejo de excepciones.

## Archivos principales
- `Printer.java`: Lógica de la impresora.
- `Document.java`: Representa un documento a imprimir.
- `PrinterException.java`: Excepción personalizada para errores de la impresora.
- `Main.java`: Menú interactivo para probar la impresora.

## Documentación
- `enunciado_resolucion.md`: Enunciado y resolución detallada.
- `diagrama_uml.md`: Diagrama UML de las clases.
- `teoria_conceptos_POO.md`: Conceptos de POO aplicados.

## Ejecución
Compila los archivos Java en la carpeta `src` y ejecuta la clase `Main`.

```bash
cd src
javac *.java
java Main
```

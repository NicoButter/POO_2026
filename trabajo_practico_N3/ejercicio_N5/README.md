# Ejercicio N°5 - Trabajo Práctico N°3

## Enunciado
La clase `PersonManager` gestiona objetos `Persona` almacenados en un `Vector` y proporciona métodos para agregar, eliminar, buscar y persistir personas usando serialización.

## Archivos principales
- `src/Persona.java`
- `src/PersonManager.java`
- `src/TestPersonManager.java`

## Ejecución
Compilar todos los archivos en la carpeta `src` y ejecutar `TestPersonManager`.

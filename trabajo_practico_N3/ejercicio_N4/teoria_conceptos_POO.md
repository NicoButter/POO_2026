# Conceptos de POO enfatizados en el ejercicio 4

- **Encapsulamiento:** Las clases `Persona` y `ArchivoPersonas` encapsulan sus atributos y exponen solo los métodos necesarios para interactuar con los datos.
- **Abstracción:** Se abstrae el manejo de archivos binarios en la clase `ArchivoPersonas`, ocultando los detalles de lectura/escritura al usuario del menú.
- **Modularidad:** Separación clara entre la lógica de datos (`Persona`), la lógica de almacenamiento (`ArchivoPersonas`) y la interfaz de usuario (`MenuArchivoPersonas`).
- **Reutilización:** Se reutilizan y extienden las clases del ejercicio anterior, agregando nuevas funcionalidades sin modificar la estructura base.
- **Responsabilidad única:** Cada clase tiene una responsabilidad bien definida.

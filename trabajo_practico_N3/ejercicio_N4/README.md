# Ejercicio N°4 - Trabajo Práctico N°3

## Enunciado
Al ejercicio anterior agregar los siguientes métodos:
- `buscar`: permite obtener los datos de una persona utilizando como clave de búsqueda el DNI.
- `alturaPromedio`: calcula la altura promedio de las personas almacenadas en el archivo.
- `alturaMayorA`: genera el archivo “auxiliar.dat” con aquellas personas cuya altura supera 1.60 mts.

## Archivos principales
- `src/Persona.java`
- `src/ArchivoPersonas.java`
- `src/MenuArchivoPersonas.java`
- `src/TestPersonas.java`

## Ejecución
Compilar todos los archivos en la carpeta `src` y ejecutar `TestPersonas`.
